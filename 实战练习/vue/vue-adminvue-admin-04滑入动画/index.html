vue-admin-04滑入动画

本文最后更新于：August 16, 2021 pm

一、改变el-main界面

二、封装一个AppMain组件

  1. 在router-view中加入动画

  2. 选择动画模式渐入渐出mode="out-in" mode 只适用于一个transition内同标签的内容，比如都是button

  3. 对动画的理解，过渡模式是建立在多组件切换模式的基础上的，也就是动态组件的切换,并且当被切换的元素是2个相同的标签（比如这里都是button），需要加key进行区分。

  4. 否则由于vue机制的原因，为了性能会使用同一个元素，也就是说你切换是不会换掉整个元素，而是替换元素中内容的值，所以需要添加key为了防止只修改内容而不重新渲染
